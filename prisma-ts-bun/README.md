# prisma-starter

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

To get started
add your schema in schema.prisma file

and run the following commands to get a working generated postgreSQL client

```bash
npx prisma migrate dev
```

```bash
npx prisma generate
```


This project was created using `bun init` in bun v1.2.3. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
